# Calculator-Web-App
A responsive Calculator Web Application built using HTML, CSS, and JavaScript. The application performs basic arithmetic operations through an interactive user interface and demonstrates front-end web development concepts, DOM manipulation, and event handling.
## Overview
A simple and responsive calculator built using HTML, CSS, and JavaScript. It provides an intuitive interface for performing basic mathematical calculations.

## Features
- Responsive user interface
- Basic arithmetic operations
- Clean and modern design
- Interactive button-based input
- Fast and user-friendly experience

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Concepts Covered
- DOM Manipulation
- Event Handling
- Responsive Web Design
- JavaScript Functions
- User Interface Design

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Start performing calculations.
